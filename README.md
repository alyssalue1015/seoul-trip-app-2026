# 2026 首爾家族制霸之旅 App

這是一個手機優先的旅行 PWA，可直接放到 GitHub Pages、Netlify、Vercel 或任何靜態網站空間。

## 功能

- 7 天 6 夜每日行程
- 每個地點的 Naver Map 搜尋導航
- 女兒 NT$10,000 購物預算記帳
- 全家吃喝玩樂記帳
- 韓元換台幣估算匯率
- 每天韓語小抄，含韓語發音與英文發音
- 小抄可直接播放韓語與英文語音
- 支援加入手機主畫面
- 視覺偏好記錄：網站設計以藍色、藍綠色系為主，未來新增頁面或製作其他網站時也優先沿用這個方向。

## 手機使用

1. 將這個資料夾部署到 HTTPS 網址。不要使用 `localhost` 作為旅行正式網址。
2. 用手機 Safari 或 Chrome 開啟網址。
3. iPhone：按分享，選「加入主畫面」。
4. Android：按選單，選「安裝應用程式」或「加入主畫面」。

記帳資料會存在各自手機的瀏覽器裡，不會上傳到網路。

## 出國也能開的做法

本機網址如 `http://localhost:4173` 或 `http://192.168.x.x:4173` 只適合同一個 Wi-Fi 測試。筆電關機、手機換網路、人在首爾時，這些網址都會失效。

正式旅行使用建議二選一：

1. Netlify：把整個 `seoul-trip-app` 資料夾拖到 Netlify Drop，取得一個 HTTPS 網址，最快。
2. GitHub Pages：把專案推到 GitHub，開啟 Pages，適合之後持續更新。

取得 HTTPS 網址後，先用手機開一次並加入主畫面。PWA 會快取 app 檔案，之後網路不穩時仍可看到已快取的行程；Naver 導航與語音播放仍可能需要手機當下有網路或系統語音支援。
